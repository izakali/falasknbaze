# FALASK Website

Bienvenue sur le projet de site web de FALASK, une entreprise dédiée à la distribution d'eau gratuite via une application mobile.

## Structure du projet

Le projet est organisé comme suit :

```
falask-website
├── src
│   ├── index.html          # Page d'accueil du site
│   ├── styles              # Dossier contenant les fichiers CSS
│   │   ├── main.css        # Styles principaux du site
│   │   ├── header.css      # Styles pour l'en-tête
│   │   └── footer.css      # Styles pour le pied de page
│   ├── scripts             # Dossier contenant les fichiers JavaScript
│   │   ├── main.js         # Script principal pour la page d'accueil
│   │   ├── app.js          # Logique principale de l'application mobile
│   │   └── utils.js        # Fonctions utilitaires
│   ├── assets              # Dossier pour les ressources
│   │   └── fonts           # Dossier contenant les fichiers de police
│   └── pages               # Dossier pour les pages supplémentaires
│       ├── about.html      # Page "À propos"
│       ├── services.html   # Page des services
│       └── contact.html     # Page de contact
├── package.json            # Fichier de configuration pour npm
└── README.md               # Documentation du projet
```

## Installation

1. Clonez le dépôt :
   ```
   git clone <URL_DU_DEPOT>
   ```

2. Accédez au dossier du projet :
   ```
   cd falask-website
   ```

3. Installez les dépendances :
   ```
   npm install
   ```

## Utilisation

Pour démarrer le projet, ouvrez le fichier `src/index.html` dans votre navigateur. Vous pouvez également configurer un serveur local pour un développement plus fluide.
